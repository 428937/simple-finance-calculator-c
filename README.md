# simple-finance-toolkit-c
A command-line financial calculator written in C. This code provides common financial calculations such as compound interest, loan EMI, present value, and future value of regular deposits.


# Features:

1) Compound Interest Calculator
2) Loan EMI Calculator
3) Present Value Calculator
     1. Single future amount
     2. Annuity (regular payments)
4) Future Value of Regular Deposits

It also has simple CLI menu interface, input validation, ANSI terminal colors for better readability

# Requirements

C compiler (GCC / Clang),
Terminal supporting ANSI colors,
math library (-lm)

# Compilation

Compile using:

gcc main.c -lm -o simple-finance-toolkit-c
